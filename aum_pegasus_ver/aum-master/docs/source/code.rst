Code documentation             
============================           
               
All the examples assume that you have set the PYTHONPATH environment variable          
to the installation location of aum, and the PATH to the aum directory.                
Run the following commands from inside the aum directory               
               
.. sourcecode:: bash           
               
    $ export PYTHONPATH=$PYTHONPATH:`pwd`/install/lib/python2.7/site-packages          
    $ export PATH=$PATH:`pwd`
